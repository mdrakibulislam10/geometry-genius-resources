1. What is DOM and what is the purpose of DOM?

Document object model(DOM) is web development programming interface. Where the code of the look like HTML is remains on object. This look-like HTML code and object is a type of model. Html files are called documents. DOM object is look like a tree.
DOM helps make website interactive. HTML elements can be get from Dom and styled with JavaScript. HTML elements and attribute can be created and removed by js with the help of dom. dom have different kind of event, such as: click, mouseover, mouseenter, etc.

2. How will you select HTML elements using DOM? Name the DOM methods.

Document contain all the HTML code. HTML elements can be accessed from this document with id-name(select only one element), class-name(select multiple elements), tag-name(select elements by tag name) and for selecting like CSS selector- query-selector(select only the first element with class or tag name) and query-selector-all(select all elements with class or tag name).
There are five methods for selecting HTML elements :

  • 1. getElementById()
  • 2. getElementsByClassName()
  • 3. getElementsByTagName()
  • 4. querySelector()
  • 5. querySelectorAll()

3. What is event bubble?

The code can be reduce by using event bubble. If the parent element provides a handler, its child elements receive that event handler. That event first capture the parent element then, capture child elements. But when event are active then, child element get the event priority first. This priority of events getting from bottom to top it's called the event bubble.

4. What is a callback function and why will you use it?

Taking a function as an argument to another function is called callback function. This function can get return value from another function and can passed any value by return. Callback function can apply all javascript methods. condition, loop, array, etc can declare and apply on this function. A callback function is just a regular javaScript function that is passed as an argument to another function.
